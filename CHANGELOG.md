# Changelog
All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-22
### Added
- Initial setup with Node.js, Express, and TypeScript.
- Configured `http-proxy-middleware` for all planned Corvus microservices.
- Implemented global JWT authentication middleware (Zero-Trust architecture).
- Added security layers: Helmet, Cors, Rate-Limiting, XSS clean, Mongo Sanitize.
- Environment configuration mapped for downstream services.
